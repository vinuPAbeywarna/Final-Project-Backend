# Vins Artisan Market

## Setup

1. Install PHP dependencies.
> composer install

2. Install NPM dependencies.
> npm install

3. Compile frontend.
> npm run development

4. Generate JWT secret.
> php artisan jwt:secret

5. Migrate & Seed the DB
> php artisan migrate --seed

All things are ready! you can start the application by:
> php artisan serve
